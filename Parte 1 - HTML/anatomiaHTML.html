Neste site irei colocar algumas informações sobre as TAGs do HTML

TAG "br"

A tag "br" tem como objetivo quebrar a linha de um texto.

TAG "input"

Esta tag permite a entrada de dados do usuário, seja em uma caixa de resposta
ou em varias respostas para selecionar ou desmarcar

TAG "hr"

Essa TAG gera uma linha horizontal.
